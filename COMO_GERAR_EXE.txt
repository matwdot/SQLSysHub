🚀 COMO GERAR EXECUTÁVEL DO SQL SYSHUB
==========================================

📋 PASSOS RÁPIDOS:

1. INSTALAR DEPENDÊNCIAS:
   pip install -r requirements_build.txt

2. EXECUTAR BUILD:
   
   Windows:
   --------
   build.bat
   
   Linux/Mac:
   ----------
   ./build.sh
   
   Manual:
   -------
   python build_exe.py

3. RESULTADO:
   - Executável criado em: dist/SQLSysHub.exe
   - Script de instalação: dist/instalar.bat
   - Instruções: dist/LEIA-ME.txt

📦 DISTRIBUIÇÃO:

Opção 1 - ZIP:
- Comprimir pasta 'dist' inteira
- Enviar ZIP para usuários
- Usuários executam SQLSysHub.exe

Opção 2 - Instalador:
- Usuários executam instalar.bat como Admin
- Programa instalado automaticamente
- Atalhos criados na área de trabalho

⚠️  REQUISITOS:

Sistema de Build:
- Python 3.7+
- PyInstaller (pip install PyInstaller)
- PyQt5
- qtawesome

Comando PyInstaller:
- Usar: python -m PyInstaller
- NÃO usar apenas: pyinstaller

Sistema de Destino:
- Windows 7+
- Visual C++ Redistributable (se necessário)
- ODBC Driver 17 (para SQL Server)

🐛 PROBLEMAS COMUNS:

"PyInstaller não encontrado":
→ pip install PyInstaller

"Módulo não encontrado":
→ pip install -r requirements_build.txt

"Executável não funciona":
→ Testar em máquina sem Python
→ Instalar Visual C++ Redistributable

"Antivírus bloqueia":
→ Adicionar exceção para SQLSysHub.exe

📞 SUPORTE:
- Verificar BUILD_INSTRUCTIONS.md para detalhes
- Logs de erro em build/
- Testar sempre em máquina limpa

✅ SUCESSO:
Executável funcionando standalone de ~50-100MB
Sem necessidade de Python no computador de destino
